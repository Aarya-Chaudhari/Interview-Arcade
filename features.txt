PROJECT: Interview Arcade

Feature 1: Tic Tac Toe + OOP Quiz
Description:
Create a two-player tic tac toe game. After the game ends, the losing player must answer 5 OOP MCQs ( provide option - attempt in cpp / javascript / python and based on user select procide question layout)

Expected Output:
- Functional 3x3 grid
- Detect winner or draw
- Show OOP quiz after game ends
- Display score

Testing:
- Check win conditions
- Check quiz trigger
- Validate MCQ answers


Feature 2: Memory Card Game
Description:

Expected Output:
- Flip animation

- Lock matched cards

Testing:
- Matching logic
- Flip behavior
- Game completion
