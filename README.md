# Interview Arcade

Interview Arcade is a gamified platform designed to help users prepare for technical interviews through interactive mini-games.

## Objective
To make interview preparation engaging by combining learning with gameplay.

## Tech Stack
- HTML
- CSS
- JavaScript
- localStorage (for data persistence)

## Project Structure
- index.html (Home Page)
- modules.html (List of modules)
- scripts/ (Game logic files)
- styles/ (CSS files)
- assets/ (Images and icons)

## Modules

1. Tic Tac Toe + OOP Quiz
2. Memory Card Game
3. Aptitude Quiz
4. Resume Keyword Matcher
5. SQL Fill in the Blanks
6. Image Riddles
7. DSA Match the Following
8. Debug the Code
9. Feedback System
10. Dashboard

## How to Run

1. Download or clone the repository
2. Open index.html in browser
3. Navigate to modules page
4. Implement each module individually

## Instructions for Participants

- Each module should be implemented separately
- Maintain clean UI and proper structure
- Use JavaScript for logic
- Do not use backend or database
- Store required data in localStorage

## Contribution Guidelines

If submitting a PR:
- Mention which module you worked on
- Describe your implementation
- Ensure code is clean and working
